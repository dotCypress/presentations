Geospatial Indexes

We need data field


location: [x, y]

location: {x:1, y:2}

and index


db.atms.ensureIndex( { 'location' : '2d' } )

Queries

  • Proximity queries
  • Bounded queries
  • Exact queries

Proximity queries


db.atms.find({location: { $near: [-70, 40]}})
						

db.runCommand({geoNear: "atms", near: [x, y]})

Distance Queries


db.atms.find({
	location: { $near: [-70,40], $maxDistance : 20 }
})
						

db.runCommand({
	geoNear: "atms", near:[x,y], $maxDistance : 20
})

Spherical Geometry

  • find(..$nearSphere..)
  • find(..$centerSphere..)
  • geoNear(..{ spherical: true }..)

Distance Calculation

The radius of the Earth is approximately:

  • 3963.192 miles
  • 6378.137 kilometers.

Distance Multiplier


db.runCommand({
	geoNear: "atms",
	near: [ -74, 40.74 ],
	spherical: true,
	distanceMultiplier: 6378.137
})

Bounded Queries

  • Circles
  • Rectangles
  • Polygons

Circles


db.atms.find({"location":
	{"$within": {"$center": [ [-74, 40.74], 10 ]}}
})

Rectangles


db.atms.find({"location":
	{"$within": {"$box": [ [-74, 40.74] ]}}
})

Polygons


db.atms.find({"location":
	{"$polygon": {"$box": [ [ 0, 0], [3, 3], [6, 0] ] }}
})

Exact queries


db.atms.find({"location": [ [-74, 40.74] ]})
